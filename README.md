# 📂 Projects

## 🏛 ACT:ON - 국민 참여형 법안 피드백 대시보드

국회 Open API를 활용해 법안 정보를 제공하고, 사용자가 법안에 대해 의견을 남길 수 있는 국민 참여형 웹 서비스.

### 주요 기능
- 국회 Open API 기반 법안 목록 및 상세 정보 조회
- 공포 법안 / 심사 중 법안 분류 제공
- 법안 검색 및 필터 기능
- 법안별 찬반 투표 기능
- 댓글 및 토론방 기능
- 마이페이지에서 참여 중인 토론 확인

### Tech
- Frontend : React
- Backend : FastAPI
- Database : MySQL
- Tools : Figma · GitHub · ERD Cloud

🔗 GitHub Repository

## ⚽ We Play - 스포츠 커뮤니티 웹 서비스

스포츠 팬들이 경기 정보 확인, 커뮤니티 활동, 실시간 채팅을 한 곳에서 이용할 수 있는 스포츠 커뮤니티 웹 서비스.

### 주요 기능
- 회원가입 / 로그인 / 로그아웃
- localStorage 기반 로그인 유지
- 게시글 작성, 조회, 수정, 삭제
- 이미지 업로드 및 미리보기
- 댓글 / 대댓글 기능
- Socket.io 기반 실시간 채팅
- TheSportsDB API 기반 KBO 경기 일정 조회
- 마이페이지 및 프로필 이미지 변경

### Tech
- Frontend : HTML · CSS · JavaScript
- Backend : Node.js · Express
- Database : Railway MySQL
- Real-time : Socket.io
- File Upload : Multer
- Deploy : Cloudflare Pages · Render

🔗 GitHub Repository
